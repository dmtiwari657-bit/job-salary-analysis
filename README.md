# Job Salary Analysis

## 📌 Project Overview
This project focuses on analyzing a job salary dataset to identify the key factors that influence salary. The analysis is performed using data preprocessing, exploratory data analysis (EDA), and visualization techniques.

## 🎯 Objective
To understand how factors like experience, education, skills, job role, and company size affect salary.

## 📊 Features of the Project
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of trends and patterns  
- Insights on salary variation  

## 📁 Dataset
The dataset contains information such as:
- Experience Years  
- Education Level  
- Skills Count  
- Job Title  
- Company Size  
- Remote Work  
- Salary  

## 📈 Key Insights
- Salary increases with experience  
- Higher education leads to better salary  
- More skills are associated with higher pay  
- Salary varies across job roles and company size  

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  

## 📌 Conclusion
The analysis shows that salary is influenced by multiple factors including experience, education, and skills. Visualization helps in understanding trends and patterns effectively.
